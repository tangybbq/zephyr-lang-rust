Documentation Index

The documentation for the Rust documentation for Rust on Zephyr is divided into two parts.

  • zephyr crate Documentation The main documentation for interfacing with Zephyr from Rust code.
  • zephyr_build support Documentation Documentation for build-time utilities, available from build.rs, primarily for allowing compile-time determination of the Zephyr configuration (e.g. conditional compilation based on Kconfig or DeviceTree values).
